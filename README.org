手数料や割引きなどで、まぎらわしいのは省略。
消費税は表を見にくくするので、税別の料金にしてある。
"データ" や "クーポン" などは、できるだけ "パケット" に置き換える。
"LTE" は "4G" と書いてもいいことになってる。
"キャリア" は通信会社のこと。

音声通話付きの契約だと、本人確認書類をコピーされる。
店舗に行っても避けられない。
コピーしとかないと、キャリアが裁判で困るらしい。
解約しても 3 年間は保存される。

* 通話アプリ

110/119 にアプリから通報はできない。 
音声通話の SIM を契約するか、最寄りの警察署や消防署に直接通報するしかない。
"緊急通報ナビ" と "火事 救急通報ナビ" アプリは、最寄りの警察署や消防署を表示してくれる。
# 緊急通報は、 "事業用電気通信設備規則" とかで定められてる。
# 管轄機関の設備に接続すること。 (0ABJ 番号しかできない)
# 位置情報とかを送信すること。
# 通報した人からは通話を終えられないように、回線を保留できること。
# 通報した人を呼び返しできること。
#
# 通話アプリは最初から、呼び返しで受ければいいのかも？
# 通報する人の操作は、電話を掛けるのと同じにしとけばいいし。
#
# Android の標準 "電話" アプリ v9.0 は、緊急通報の時に現在地を表示する。
# Android 6 以降の Nexus, Android One, Pixel のみ。
# 110, 119, 118 限定。

# 認証の時に音声通話や SMS 、または SNS を使うのがある。
# 通話アプリじゃないけど、キャリアメール (MMS) を要求する銀行とかもある。

** 中継電話

音声通話の SIM を契約しないと使えない。
電話番号の前に番号を追加して発信するだけのもの。 (プレフィックス)
音声通話のパケットは優先されるので、途切れにくい。
パケットを消費しない。
相手に通話料の負担がない。 (IP 電話のパケットを除く)

アプリで着信はできない。 (デフォルトアプリにならない)
着信通知とかから直接電話を折り返すと、中継されないので通常料金になる。 *(!?)*
不便だけど通知や履歴を使わないか、非公式アプリでなんとかする。

+通話定額に必須のアプリ。+
通話定額で、設定変更がいらないアプリ。
格安キャリアごとに、似たアプリがあったりなかったり。

# ガイドラインは "モバイルサービスの提供条件・端末に関する指針" になった。 (2017/01/10)
# "SIM ロック解除の円滑な実施に関するガイドライン" として含まれてる。
# 契約してない人の相手をしろとは書いてない。

解除しないと違う回線に移るとき、スマホを使い回せない。
2015/05/01 より前に買ったスマホは、解除できないことになってるのが多い。

解除なしで、同じ回線を使う格安キャリアに移れるスマホはある。
ドコモのスマホは解除なしで使える。
au も 4G 対応してて VoLTE 非対応なら解除なしで使える。
ソフトバンクは iPhone だけ解除なしで使える。

ドコモがガイドラインを履行。 (前倒しで 2017/05/24 から)
対象は 2015/05/01 以降に発売されたスマホ。
ただし、ドコモと契約して買った本人しか解除できない。
本人であっても、解約から 101 日以降は解除できない。
(それ以前の解除対象スマホは、ドコモと契約してる人なら解除できる)

au がガイドライン履行について告知。 (2017/06/30)
対象は 2015/04/23 以降に発売されたスマホ。
ただし、 au と契約して買った本人しか解除できない。 (2017/12/01 から)
本人であっても、解約から 101 日以降は解除できない。 (2017/12/01 から)
(2017/07/31 までの au VoLTE 対応スマホが、同じ回線を解除なしで使えるようになるかは不明)
