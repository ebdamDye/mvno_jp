** アプリ専用通話

同じアプリを利用する相手とだけ通話できる。

*** LINE (LINE)

暗号化され始めたメッセージアプリ。 (Letter Sealing)
全員が "Letter Sealing" を on にしてないと、ロックアイコンが出ない。
暗号化された無料通話などができる。
電話帳の送信がデフォルト。
# AMR-WB or G.723.1a-L or G.723.1a-H?

*** SkyPhone (クアッドシステム)

機能は無料通話のみ。
ユーザ登録や利用料は不要。
初回起動時に SkyPhone 専用番号が決まる。
スマホを変えると SkyPhone 専用番号も変わる。
相手の SkyPhone 専用番号を知らないと、掛けられない。
低音質に設定すれば途切れにくくなって、パケットを抑えられる。
会社は富山県にある。
# CELT

暗号化された無料通話などができる。

* SIM ロック

| 2010/06/30 | 総務省のガイドライン策定。                                                                    |
| 2014/12/22 | 総務省のガイドライン改正。                                                                    |
| 2015/05/01 | 買ってから 180 日後に解除できる。                                                             |
| 2017/01/10 | 総務省のガイドライン改正。                                                                    |
| 2017/08/01 | 割賦払いで買うと、 100 日後に解除できる。 解約の時は原則解除。 同じ回線なら解除なしで使える。 |
| 2017/12/01 | 一括払いならすぐ解除できる。                                                                  |

解除しないと違う回線に移るとき、スマホを使い回せない。
2015/05/01 より前に買ったスマホは、解除できないことになってるのが多い。

すでに解除なしで、同じ回線を使う格安キャリアに移れるスマホがある。
ドコモのスマホは解除なしで使える。
au も 4G 対応してて VoLTE 非対応なら解除なしで使える。
ソフトバンクは iPhone だけ解除なしで使える。

ガイドラインには 3G から 4G へ移行しろとも書いてある。
au は最初から、格安キャリアで 3G のパケット通信ができない。
ソフトバンクもすでに、 3G の一部を終了すると告知してた。
最後まで残しそうなのは、例によってドコモかな。
具体的な告示とかはないけど、何年かしたら 3G は使えなくなるのかも。
# 2G 終了の 2012 年は地デジ移行もしてる。
# 海外の 2G は 2017 年も終了の途中。
#
# au の設備では、 3G のパケット通信を格安キャリアに送れない。
# 格安キャリアのエリアが狭くなるけど、au は設備をそのまま使ってる。
# 清々しいほど頑なに。

3G がなくなると、当方のガラケーは使えなくなる。
